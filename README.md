# RealityPrompt Engine

Générateur de prompts éditoriaux pour créer du contenu authentique sur les réseaux sociaux. Anti-bullshit, ancré dans la réalité.

Développé par **[@JOJODEVS-GIT](https://github.com/JOJODEVS-GIT)**

---

## Fonctionnalités

- Génération de prompts optimisés pour LinkedIn, Instagram, Facebook, X
- Choix de ton (calme, cash mesuré, cash frontal) et d'intensité
- Thèmes : discipline, résilience, doute, progression silencieuse...
- Option contexte africain (contraintes réalistes)
- Dark mode intégré
- Copier/coller en un clic
- 100% standalone — un seul fichier HTML, aucune dépendance

## Utilisation

Ouvrir `generatreur de prompt.html` dans un navigateur. C'est tout.

## Stack

- **HTML / CSS / JavaScript** — Vanilla, zéro framework
- Design responsive avec CSS variables
